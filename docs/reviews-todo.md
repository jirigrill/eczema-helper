### Phase 6 — Notifications
- [ ] **Timezone handling:** The cron scheduler uses `HH:MM` matching but does not account for server timezone vs user timezone. If the server is UTC and user is CET (UTC+1), a 20:00 reminder fires at 19:00 local time. Either document the assumption (same timezone) or implement timezone-aware scheduling.
